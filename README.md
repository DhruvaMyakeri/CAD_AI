# CAD_AI

Pre-Analysis Engineering Intelligence Platform

## What It Does

- Extracts geometry from STEP files
- Infers structural behavior (pre-FEA)
- Evaluates fatigue margins
- Estimates precision difficulty
- Optimizes manufacturing process
- Recommends materials
- Generates executive engineering report (LLM)

## Run

```bash
python master_engine_v5.py
```
